Electric Vehicle (EV) Dashboard (Tableau)

A Tableau visual analytics project exploring EV adoption and market patterns through interactive charts, filters and geographic exploration.

Problem

Build an interactive dashboard that helps users explore EV trends and answer questions like: How is EV adoption changing over time? Which segments are growing fastest? What patterns appear by region or category?

Data

  • Dataset: EV-related dataset
  • Core fields: date/year, vehicle type, manufacturer/model, region, metrics (counts, shares)
  • Preparation: cleaning, handling missing values, creating calculated fields for analysis

Methods

  • Tableau: interactive dashboard design with filters and parameter controls
  • Visuals: time-series trends, breakdowns by category/make/model, maps
  • Analytics: calculated fields, groupings, highlighting top contributors and changes over time

Results

  • Delivered an easy-to-navigate dashboard for exploring EV trends and comparisons
  • Enabled rapid identification of growth segments and high-contributing categories
  • Provided stakeholder-friendly visuals suitable for presentations or portfolio review
